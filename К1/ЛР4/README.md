Программа работает с CSV-файлом, который заранее создаётся пользователем с именем data.csv и содержит строки с числовыми значениями. При запуске функция пытается открыть файл и построчно обрабатывает его содержимое. Для каждой строки выполняется преобразование данных к нужным числовым типам, а корректные строки учитываются в общем счётчике. Если файл отсутствует или строка имеет неправильный формат, ошибка обрабатывается без остановки программы. В конце работы выводится отчёт о количестве успешно обработанных строк.
