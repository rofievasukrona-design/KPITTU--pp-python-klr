Краткое описание
- `generate_data.py` — скрипт генерации тестовых данных (`data.txt`) с дневными и ночными температурами.
- `main.py` — анализатор: читает `data.txt`, вычисляет скользящую среднюю по 7 дням, находит аномалии (день/ночь > 10°C), определяет самую длинную серию роста дневной температуры.
Как запустить:

```bash
python generate_data.py    
python main.py             
```

Примечания
- `generate_data.py` создаёт файл `data.txt` в текущей директории; убедитесь, что `main.py` запускается из той же папки или что `data.txt` доступен для чтения.
- Если при запуске `main.py` появляется сообщение об ошибке, запустите сначала `generate_data.py`.
