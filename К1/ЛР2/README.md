task_01.py

Программа получает два числа и формирует диапазон от меньшего к большему. Она перебирает каждое значение, пропуская чётные через continue, и накапливает сумму всех нечётных. Финальный результат выводится на экран.

task_02.py

Перед началом цикла необходимо проверить размер входной последовательности. Если последовательность содержит менее двух элементов (т.е. она пуста или состоит из одного числа), процесс поиска возрастаний невозможен, и программа должна сразу вернуть результат 0. Программа последовательно обходит список чисел, начиная со второго элемента. На каждом шаге она сравнивает текущее значение с предыдущим, который хранится в переменной prev. Если текущее число больше предыдущего, счётчик увеличивается. После сравнения prev обновляется, и процесс продолжается до конца последовательности. Итоговое количество возрастаний выводится пользователю.

task_03.py

Программа принимает строку и проходит по каждому символу, оставляя только буквы и пробелы. Все допустимые символы сразу преобразуются к нижнему регистру. Полученный очищенный текст формируется в единую строку, после чего разбивается на отдельные слова. Результат выводится в виде очищенной строки и списка слов.
