Генератор `sliding_sum(data, k)` возвращает суммы элементов скользящего окна длины `k` по входной последовательности, используя `yield` (лениво, O(n)).

Требования:
- проверка `k` (тип, положительность, размер)
- корректная остановка генератора
- ленивость (yield)

Пример использования:

```python
from logic import sliding_sum
print(list(sliding_sum([1, 2, 3, 4], k=2)))  # [3, 5, 7]
```
