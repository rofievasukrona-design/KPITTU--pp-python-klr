Алгоритм: Сумма скользящего окна

Описание:
- Функция `max_sum(arr, k)` вычисляет максимальную сумму любого непрерывного подмассива длины `k`.
- Используется метод "скользящее окно": сначала считаем сумму первых `k` элементов, затем последовательно вычитаем самый левый элемент окна и добавляем следующий элемент массива, обновляя максимальную сумму.
- Время выполнения: O(n), память: O(1).

Проверка входных данных:
- Если `k` не является положительным целым числом, или `k` больше длины массива, функция возбуждает `ValueError`.

Как запускать тесты:

В командной строке, находясь в папке проекта (где лежат `sliding.py` и `test_sliding.py`), выполнить:

```
python -m unittest ЛР7.test_sliding
```

Или запустить конкретный файл теста:

```
python ЛР7/test_sliding.py
```
