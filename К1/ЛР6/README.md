task_01.py
Программа принимает вложенный список чисел произвольной глубины и последовательно обрабатывает каждый элемент. Если элемент является списком, функция вызывает сама себя для подсчёта суммы внутри этого списка. Если элемент является числом и имеет положительное значение, он добавляется к общей сумме. Отрицательные числа и другие типы данных игнорируются. В результате возвращается сумма всех положительных чисел из всей структуры.

task_02.py
Программа принимает массив чисел и заданную длину подмассива k. Сначала проверяется, что размер массива достаточен для выполнения задачи. Затем используется метод скользящего окна: вычисляется сумма первых k элементов и далее окно последовательно сдвигается на один элемент, обновляя сумму без пересчёта всего подмассива. При каждом шаге фиксируется максимальная сумма и позиция её начала. В результате возвращается подмассив длины k, сумма элементов которого является наибольшей.
