task_01.py
Программа принимает текст и удаляет из него все символы, кроме букв и пробелов, одновременно приводя строки к нижнему регистру. Полученный текст разбивается на слова, после чего строится частотный словарь. Из него выбираются десять самых распространённых слов и формируется отдельный набор слов длиной больше шести, встречающихся чаще остальных. Далее вычисляется доля каждого слова в процентах относительно общего числа слов. Все рассчитанные данные выводятся пользователю.

task_02.py
Функция принимает несколько наборов чисел и параметры обработки. Для каждого набора она рассчитывает сумму, среднее значение и медиану. При включённой опции merge все данные объединяются и анализируются как единый массив. Если задан normalize, значения внутри каждого набора преобразуются к диапазону от 0 до 1. Когда указан параметр round, все вычисленные показатели округляются до нужного количества знаков. Результат возвращается в виде структуры, содержащей рассчитанные статистики для одного или нескольких наборов.
